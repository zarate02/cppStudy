# DAY 1

## 어셈블리 언어
- IDE(통합환경) : SASM
- 어셈블러 : 개발언어 => 기계어

## 컴퓨터
- CPU (ALU)
- 메인메모리 (RAM)
- 하드디스크 (HDD)

## 비트바이트
- 비트 : 0,1
- 바이트 : 8비트
- 8비트 = 1바이트

## 2진수
- 0b0, 0b1, 0b11, 0b100

## 16진수
- 0x0A, 0x1001

## 레지스터
- R + A,B,C,D + X : 64bit
- E + A,B,C,D + X : 32bit
- A,B,C,D + X : 16bit
- A,B,C,D + L : 8bit
- mov A, B : B에 있는 값을 A로 옮김

## .data 변수선언사용
- section .data
- [변수이름] [크기] [초기값] 
- 크기단위 : db(1), dw(2), dd(4), dq(8)

## .bss 변수선언사용
- section .bss
- [변수이름] [크기] [개수]
- 크기단위 : resb(1), resw(2), resd(4), resq(8)

## mov의 비밀
- 일반적인 mov 는 주소를 넣어준다
- mov rax, [a] 는 a값을 넣어준다
- move [a], byte 0x55 : 변수에 상수넣을때 크기를 명시해야된다
- move [a], cl : 변수에 레지스터값을 넣을때

## 문자와 엔디안
- [변수이름] [크기] [문자열] [null]
- 리틀엔디안 : 뒤집혀서저장 (인텔, AMD환경) : 캐스팅유리
- 빅엔디안 : 실제순서대로저장 : 비교유리

## 사측연산
- GET_DEC 사이즈 넣을곳 : 값입력
- PRINT_DEC 사이즈 출력값 : 값출력
- ADD 레지스터or[메모리] 레지스터or[메모리] : 더하기
- ADD [메모리] 크기 상수 : 더하기
- MUL 레지스터 : 곱하기
- DIV 레지스터 : 나누기
